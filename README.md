# etymology-english

「連想式でみるみる身につく語源で英単語」の接頭辞、接尾辞、語幹を表にまとめたものです。色んな形式で後で遊べるようにclojureでデータ持たせているだけ。

## Usage

`platex`や`omake`の入った状態で`lein run`すると`main.pdf`ができあがります。
